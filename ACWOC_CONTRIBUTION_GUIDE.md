# ACWOC Contribution Guide

Welcome! Thank you for your interest in contributing to StegaSnoop.

This repository is beginner-friendly and part of ACWOC.

## How to Contribute
1. Fork this repository
2. Clone your fork
3. Create a new branch for your changes
4. Pick one open issue
5. Make your changes
6. Commit with a clear message
7. Push your branch
8. Open a Pull Request and mention the issue number

## Contribution Rules
- Work on one issue at a time
- Keep code simple and readable
- Add comments where necessary
- Do not push directly to the main branch

## Beginner Issues
Look for issues labeled:
- acwoc
- easy

Happy contributing!
